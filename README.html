✅ TodoApplication

A simple To-Do List application built with React and a backend API for managing tasks.

🚀 Features

  • 📝 Add new tasks with a title and description.
  • ✅ Mark tasks as completed or pending.
  • ✏️ Edit tasks dynamically.
  • 🗑️ Delete tasks when no longer needed.
  • 🎨 Modern UI with Tailwind CSS styling.

🛠️ Tech Stack

  • Frontend: React, Tailwind CSS
  • Backend: Node.js / Spring Boot
  • Database: MySQL / MongoDB

⚙️ Setup & Installation

1. Clone the Repository

git clone https://github.com/Jimmyvrushabh/TodoApplication.git
cd TodoApplication

2. Install Dependencies

npm install

3. Start the Development Server

npm run dev

The app will be running at http://localhost:5173.

🚀 Deployment

To deploy the project on GitHub Pages, Vercel, or Netlify, follow the respective deployment guides.
